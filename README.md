# 基于springboot的二手物品交易管理系统6

#### 介绍
基于springboot的二手物品交易管理系统6拥有多种角色

- 管理员：菜单管理、分类管理、角色管理、公告管理、评论管理、求购物品信息管理、物品管理、学生管理、用户管理等

- 用户：发布商品、发布求购信息、分类查询、登录注册、个人后台等

- 其它角色：管理员可以设置多种角色
